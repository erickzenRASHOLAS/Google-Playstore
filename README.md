# Google Playstore

Este repositorio contiene recursos relacionados con un proyecto de análisis y/o recopilación de datos de Google Playstore. Incluye scripts, datos y documentación para ejecutar tareas de scraping, análisis y visualización.

## Requisitos

- Python 3.8+ (si hay scripts en Python)


1. Clona el repositorio:

   git clone <repo-url>

2. Entra en la carpeta del proyecto:

   cd Google-Playstore

3. (Opcional) Crea un entorno virtual e instala dependencias:

   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt

## Uso

- Describe aquí cómo ejecutar los scripts principales. Ejemplo:

  python scripts/analizar.py --input datos/app_data.csv

## Estructura

- `scripts/` — scripts útiles para extracción y análisis
- `data/` — datos de ejemplo o salidas
- `notebooks/` — notebooks de análisis
